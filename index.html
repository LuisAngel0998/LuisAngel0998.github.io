Listado de etiquetas HTML


  1. < !-- -- > : Puede usar la etiqueta de comentario para "ocultar" los scripts de los navegadores

    • Ejemplo: < !- - esto es un ejemplo -->

  2. < !DOCTYPE html> : Declaracion del tipo de documento.


  3. < a >: define un enlance que se utiliza para enlazar de una pagina a otra.

    • Ejemplo: < a href="https://www.uasd.edu.do">, enlace la pagina web de la UASD

  4. < abbr >: Esta etiqueta define una abreviatura, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

    • Ejemplo: The < abbr title="Universidad Autonoma De Santo Domingo" > UASD

  5. < acronym >: La etiqueta se uso en HTML 4 para definir un acronimo.


  6. < address >: Esta etiqueta define la informacion de contacto del autor/propietario de un documento o articulo.

    • Ejemplo:< address > escrito por < a href="ejemplo.com">Carlos Perez< /a >.
      Visitanos en:
      Ejemplo.com
      Box 564, Santo Domingi
      RD < /address >

  7. < applet >: Esta etiqueta se usó en HTML 4 para definir un subprograma incrustado (complemento).


  8. < area >: Esta etiqueta define un área dentro de un mapa de imagen (un mapa de imagen es una imagen con áreas en las que se puede hacer clic).

    • Ejemplo: < map name="workmap"> < area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
      < area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
      < area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm"> < /map>

  9. < article >: Esta etiqueta especifica contenido independiente

    • Ejemplo: < article> < h2>UASD< /h2> < p> < /p> < /article>

  10. < aside>: Esta etiqueta define algún contenido además del contenido colocado.

    • Ejemplo: < p>Juan Pablo Duarte y Díez, fue un militar, político, activista y Fundador de la República Dominicana.​​< /p>
      < aside>
      < h4>Republica Dominicana< /h4>
      < p>República Dominicana es un país del Caribe que comparte la isla La Espanola con Haití al oeste.< /p>
      < /aside>

  11. < audio>: Esta etiqueta se utiliza para incrustar contenido de sonido en un documento, como música u otras transmisiones de audio.

    • Ejemplo: < audio controls> < source src="horse.ogg" type="audio/ogg">
      < source src="horse.mp3" type="audio/mpeg">
      Your browser does not support the audio tag. < /audio>

  12. < b>: especifica texto en negrita sin ninguna importancia adicional.

    • Ejemplo: < p> yo me llamo - < b>Luis Angel Matos< /b>.< /p>

  13. < base>: especifica la URL base y/o el destino de todas las URL relativas en un documento.

    • Ejemplo: < base href="https://www.uasd.edu.do/" target="_blank">

  14. < basefont>: se uso en HTML 4 para especificar un color de texto predeterminado, un tamaño de fuente o una familia de fuentes para todo el texto en un documento HTML.


  15. < bdi>: aísla una parte del texto que podría estar formateada en una dirección diferente de otro texto fuera de ella.

    • Ejemplo: < bdi>hrefs< /bdi>:

  16. < bdo>: se utiliza para anular la dirección del texto actual.

    • Ejemplo: < bdo dir="rtl">
      texto de ejemplo.
      < /bdo>

  17. La < big>etiqueta se usó en HTML 4 para definir un texto más grande.

  21. < button>: define un botón en el que se puede hacer clic.

    • Ejemplo: < button type="button">click aqui< /button>

  22. < canvas>: se usa para dibujar gráficos, sobre la marcha, a través de secuencias de comandos

    • Ejemplo: < canvas id="myCanvas"> algunos navegadores no soportan esta etiqueta < /canvas>
